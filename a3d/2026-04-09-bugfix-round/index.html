Fix #1789/#1792 — SessionImport crash

file_get_contents(https:///api/v1/sessions): Failed to open stream Causa: env('APP_URL') ritorna null con config cachata Fix: str_replace('https://', '', config('app.url'))

Fix #1796 — Date campagna non salvate

InputDatetimeTz emette evento 'modelSaving' ma ModelForm non ascoltava. Fix: aggiunto listener 'modelSaving' => 'onModelSaving' in ModelForm.php

Fix #1797 — Usertag campagna lento

CampaignUsertag caricava tutti i 228 tag (Usertag::has('users')) Fix: filtra per utenti con subscription nella campagna (228 -> 23 tag)

Fix #1798 — Export tag incompleto

AbstractTagBuilder::getTags() usava has('sessions') — solo 3 tag su 249 Fix: cambiato in has('users') — ora include tutti i tag con utenti

Bug #1790 — PDF con menu navigazione

DashboardProfileSubscriptionsController::pdf() passa $data['media'] = 'pdf' ma la view non controlla $media per nascondere il nav. Fix da fare: @if(($media ?? null) !== 'pdf') nel template report

Bug #1787 — Logo mancante

Media ID 65 (Screenshot-2025-04-16-alle-17.41.06.png) non trasferito durante migrazione. Il file e in storage/app/public/65/ — la directory non esiste su s0. Immagine originale su avocado (ancora acceso, snapshot 372905032).
