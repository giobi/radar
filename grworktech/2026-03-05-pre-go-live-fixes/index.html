Technical details
✓
1. Confirmation message now in English

After submitting an application, the page used to show "Candidatura inviata" in Italian. It now shows "Application submitted. Thank you for applying. Only shortlisted candidates will be contacted."

✓
2. Confirmation email sent automatically

Candidates now receive an email right after applying. It confirms their application was received, shows the position name and submission date, and includes a link to the candidate journey page.

Emails are sent from noreply@grworktech.com with proper authentication (SPF, DKIM, DMARC) so they won't end up in spam.

✓
3. Question layout improved

Previously, the question number/title and the question text ran together without clear separation. Now the title appears in bold with a line underneath, and the question text appears below in a different style. Much easier to read.

✓
4. Test data cleaned up

All 43 test applications created during development and QA have been removed from the database. The system now has a clean slate — zero submissions — ready for real candidates.

✓
5. Trade Expert URL corrected

The Trade Expert application form URL used to contain the word "facilitation" which was left over from an earlier naming. The link has been updated to forms.grworktech.com/f/cefta-trade-expert-2026 — clean and correct.

✓
6. Trade Expert Services title fixed

The title showed "Trade Expert on Trade in Services". It now reads "Trade Expert – Trade in Services" with a proper dash, matching the official vacancy notice format.

✓
7. Date format corrected

The date placeholder in the signature section showed "DD/MM/AAAA" (Italian format code). Changed to DD/MM/YYYY as expected in English.
