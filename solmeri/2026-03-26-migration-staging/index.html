Problemi risolti durante la migrazione

1. Object Cache Pro (Redis) Tutti i siti usavano Object Cache Pro con Redis di Cloudways. Le credenziali Redis erano hardcoded in wp-config.php. Rimosso il drop-in object-cache.php e le configurazioni Redis. Da riconfigurare con Redis locale post-switch. 2. Code Snippets su hvac-solmeri.com Uno snippet PHP personalizzato usava sintassi incompatibile con PHP 8.3. Plugin disabilitato temporaneamente. Snippet da verificare/aggiornare prima del go-live. 3. Forge provisioning Ubuntu version mismatch Primo tentativo fallito: Forge creato con Ubuntu 22.04, server era 24.04. Ricreato con versione corretta.

Prossimi passi

1. Verifica visuale pagine interne (Elementor layouts) 2. Fix Code Snippets su HVAC (aggiornamento plugin o snippet) 3. Setup caching: Redis Object Cache + FastCGI 4. Rimuovere plugin Object Cache Pro (licenza Cloudways) 5. Quando tutto OK: switch DNS reali + SSL 6. Tenere Cloudways vivo 1 settimana come rollback
